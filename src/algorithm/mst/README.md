# MST

최소의 비용으로 모든 노드가 ㅇ녀결된 트리

- Kruskal: 전체 간선 중 작은 것부터 연결
- Prim: 현재 연결된 트리에 이어진 간선 중 가장 작은 것을 추가
- Prim이 비교적 구현이 쉬운편이다

## 아이디어
- 간선을 인접리스트 형태로 저장한다
- 시작점을 힙에 넣는다
- 힙이 빌 때까지, 방문여부 체크, 비용 추가, 연결된 간선 새롭게 추가

## 시간복잡도
- O(lgE) - 모든 간선에 관하여 힙 추가, 힙 제거

## Tip
- MST 문제 유형: 모든 노드가 연결되도록 / 이미 연결된 노드를 최소 비용으로 줄이기
